# Telegram HTML Export Parser

Скрипт для разбора HTML-экспорта Telegram-канала (выгрузка через Telegram Desktop) в структурированные данные: JSON + Markdown.

## Когда это нужно

У тебя есть свой Telegram-канал, ты хочешь:
- Сделать контент-инвентаризацию (сколько постов, когда, какие темы)
- Обучить AI-агента (копирайтера) своему голосу — скормить ему всю историю
- Получить статистику реакций / просмотров за весь период жизни канала
- Забрать медиа из постов
- Сделать семантический поиск по своим старым постам

## Как использовать

### Шаг 1. Выгрузить канал из Telegram Desktop

1. Открой Telegram Desktop (не мобильный!)
2. Зайди в свой канал → три точки → Export chat history
3. Параметры:
   - Format: **HTML**
   - Include: текст + реакции + ссылки (медиа — по желанию, большой размер)
4. Жди завершения — Telegram выгрузит папку с `messages.html`

### Шаг 2. Запустить парсер

```bash
# Установи зависимость (один раз)
pip install beautifulsoup4

# Отредактируй пути в скрипте
# INPUT = "/путь/до/messages.html"
# OUT_DIR = "/куда/сохранить"

# Запусти
python3 parse_telegram.py
```

### Шаг 3. Получи результаты

В `OUT_DIR` появятся:
- `posts.json` — все посты структурированно (текст, дата, реакции, ссылки, медиа-url)
- `posts.md` — читаемый Markdown для просмотра / скармливания AI
- `stats.json` — сводная статистика (топ-темы, самые популярные посты, динамика)

## Что можно сделать дальше

После парсинга у тебя на руках структурированные данные. Типичные сценарии:

**Обучить Copywriter-агента твоему голосу:**
```
Прочитай posts.md. Проанализируй мой стиль письма: ритм, длина абзацев,
типичные слова, структура. Создай профиль моего голоса в knowledge/brand/voice.md.
```

**Найти свои самые залетающие посты:**
```
Открой stats.json. Покажи топ-20 постов по реакциям.
Выдели общие паттерны — что делает их успешными.
```

**Сделать серию продолжений популярных тем:**
```
Вот топ-10 моих залетевших постов. Предложи 5 продолжений каждой темы,
на которые аудитория среагирует так же или сильнее.
```

## Ограничения

- Парсит только **HTML-экспорт** — не работает с приватными каналами напрямую
- Медиа скачивать не умеет (только достаёт ссылки из HTML)
- Не работает с gif / sticker packs (по крайней мере сейчас)
- Оптимизирован под русский язык, но парсит любой
